Чтобы начать программировать на Java, нужно установить комплект разработчика приложений (SDK). Он позволит запускать, отлаживать и исполнять наши первые программы. Также нам понадобится среда разработки (IDE), где мы и будем писать код.

Установка SDK

Java Development Kit (JDK) — это комплект ПО для разработчиков приложений на Java. Стандартная установка включает в себя интерпретатор, облегчённый интерпретатор JRE, компилятор, библиотеку Java-классов, отладчик, средства архивации и сжатия jar и другое
Cкачать актуальную версию с инструкцией по установке можно на сайте:\n https://adoptopenjdk.net/installation.html

Установка IDE

IntelliJ IDEA – одна из самых мощных универсальных сред разработки. Существует две версии IntelliJ IDEA – Community и Unlimited. Первая – бесплатная, но ограниченная. Нам хватит возможностей бесплатной версии, соответственно, ориентируемся на установочный файл с именем idealC.
Для установки в ОС Windows нам потребуется скачать с официального сайта (https://www.jetbrains.com/idea/download/#section=windows) исходник с расширением exe. Далее запускаем его и следуем инструкциям установщика.

Создание первой программы

Когда с установкой и настройкой IntelliJ IDEA покончено, можно приступить к работе. Начинается она, разумеется, с создания IntelliJ IDEA проекта

Нажимаем "Create New Project"\n В поле "Project SDK" находим папку с установленной ранее JDK

В новом окне ставим галочку "Command Line App" для создания шаблона программы

Придумываем имя программы и пакета и выбираем место расположения
