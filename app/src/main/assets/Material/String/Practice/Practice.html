Задача 1

Напишите программу на Java для обращения строки
Входные данные:
  Введите строку: Быстрый коричневый лис
Ожидаемый результат :
  Обратная строка: сил йывенчирок йыртсыБ 

import java.util.Scanner;
public class Exercise1 {
     public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Введите строку: ");
        char[] letters = scanner.nextLine().toCharArray();
        System.out.print("Обратная строка: ");
        for (int i = letters.length - 1; i >= 0; i--) {
            System.out.print(letters[i]);
        }
        System.out.print("\n");
    }
}

Пример вывода: 

Введите строку: Быстрый коричневый лис
Обратная строка: сил йывенчирок йыртсыБ

Задача 2

Напишите Java-программу для подсчета букв, пробелов, чисел и других символов входной строки.
Входные данные:
  Строка: Aa kiu, я swd skieo 236587. GH kiu: sieo ?? 25,33
Ожидаемый результат :
  букв: 23
  пробелов: 9
  чисел: 10
  другое: 6 

import java.util.Scanner;
public class Exercise2 {

 public static void main(String[] args) {
		String test ="Aa kiu, I swd skieo 236587. GH kiu: sieo?? 25.33";
		count(test);
	}
	public static void count(String x){
		char[] ch = x.toCharArray();
		int letter = 0;
		int space = 0;
		int num = 0;
		int other = 0;
		for(int i = 0; i < x.length(); i++){
			if(Character.isLetter(ch[i])){
				letter ++ ;
			}
			else if(Character.isDigit(ch[i])){
				num ++ ;
			}
			else if(Character.isSpaceChar(ch[i])){
				space ++ ;
			}
			else{
				other ++;
			}
		}
		System.out.println("букв: " + letter);
		System.out.println("пробелов: " + space);
		System.out.println("чисел: " + num);
		System.out.println("другое: " + other);
			}
}

Пример вывода: 

букв: 23
пробелов: 9
чисел: 10
другое: 6
